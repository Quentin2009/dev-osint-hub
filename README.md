## 🚀 Installation

1. **Cloner le dépôt :**
   ```bash
   git clone [https://github.com/votre-organisation/votre-projet.git](https://github.com/votre-organisation/votre-projet.git)
   cd votre-projet
Installer les dépendances :

Bash
npm install
Configurer les variables d'environnement :

Bash
cp .env.example .env
Renseigne ensuite tes clés d'API et identifiants dans le fichier .env.

Lancer l'application en mode développement :

Bash
npm run dev
L'application sera accessible sur http://localhost:3000.
